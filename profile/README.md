## Engineering principles

Every engagement is built on the same foundations:

| Principle | What it means in practice |
|---|---|
| **Explainability** | Every transformation, decision, and process is documented and traceable |
| **Auditability** | Full lineage from source to output; every change attributable |
| **Privacy by Design** | PII detection and protection built into pipelines, not bolted on |
| **Immutable Audit Trail** | Append-only raw layers; never mutate source data |
| **Evidence-Based Acceptance** | Milestones backed by logs, reports, and quality scores — not assertions |
| **Human in the Loop** | Destructive or schema-altering operations require human approval |
| **Sustainable Handoff** | Clients operate independently post-engagement: runbooks, training, self-healing systems |

Full standards: engineering principles, data engineering principles, and git/repo conventions are published with every engagement.
